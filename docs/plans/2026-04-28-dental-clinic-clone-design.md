# Dental Clinic Clone Design

Goal: Build a Next.js and Tailwind landing page for a fictional Bucharest dental clinic, adapted from the visual and structural language of mazecenter.ro.

Architecture: One App Router page renders semantic sections backed by local arrays for treatments, services, team photos, articles, and testimonials. Tailwind handles layout and responsive behavior. Global CSS defines the reference-inspired theme tokens, keyframes, and reusable hard-edged visual treatments.

Implementation tasks:

1. Replace generated metadata with Romanian clinic SEO metadata.
2. Replace the generated home page with header, hero, treatment rail, manifesto, services, team/about, articles, testimonials, and footer sections.
3. Replace global CSS with the brand token layer, font stacks, motion keyframes, focus styles, and reduced-motion rules.
4. Update README with project purpose and local commands.
5. Run lint and production build.

Quality bar: flagship portfolio demo. The result should feel human-made and close to the reference site's pacing, not like a generic medical template.
