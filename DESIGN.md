# Design Notes

Reference lane: MAZE Center style adapted to dentistry.

Palette:
- Ink: #050505 for header CTA, dark sections, strong text.
- Paper: #fbfaf7 for warm page backgrounds.
- White: #fefdfb for header and high-contrast text fields.
- Coral: #ed6a5e for small link accents and active dots.
- Clinic blue: #111e70 for deep gradient sections.
- Sage: #7ba7a2 for restrained dental-wellness cues.

Typography:
- Display and headings follow Poppins-like geometry, tight tracking, medium to semibold weights.
- Body follows Open Sans-like humanist sans with generous line height.
- Button labels use Roboto-like narrow uppercase letter spacing.

Layout:
- Header is simple, white, 80px high, left logo, small menu glyph, square black appointment button.
- Hero uses centered white text over full-bleed dental clinic photography, then a wide organic product/planet-like visual band.
- Mid-page uses a black-to-deep-blue vertical gradient with three uneven text columns, not boxed cards.
- Service cards are tall image-led tiles with text overlays and plain text links.
- Blog and testimonials keep the WordPress editorial feel: lists, dates, quotes, and footer link density.

Motion:
- Soft page load fade-up, slow horizontal carousel drift, image hover zoom at 700ms cubic-bezier(0.22, 1, 0.36, 1).
- Respect prefers-reduced-motion.
